# React 3D Animation Project

Dự án React với các thư viện animation và 3D mạnh mẽ.

## Các thư viện đã tích hợp

- **@gsap/react** - GSAP React hooks
- **@react-three/drei** - Helpers cho React Three Fiber
- **@react-three/fiber** - React renderer cho Three.js
- **@studio-freight/lenis** - Smooth scrolling
- **@types/three** - TypeScript types cho Three.js
- **animejs** - Animation library
- **clsx** - Utility cho className
- **gsap** - Animation library
- **react** - React framework
- **react-dom** - React DOM renderer
- **tailwindcss** - Utility-first CSS framework
- **three** - 3D library
- **zustand** - State management

## Cài đặt

```bash
npm install
```

## Chạy dự án

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Cấu trúc dự án

```
├── src/
```
