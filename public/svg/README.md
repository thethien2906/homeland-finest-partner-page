# SVG Assets Folder

Thư mục này chứa các file SVG cho dự án.

## Sử dụng:

Các file SVG có thể được import và sử dụng trong React components:

```jsx
import MySVG from '/svg/my-file.svg'
```

Hoặc sử dụng như background image trong CSS.

## Lưu ý:

- SVG files sẽ được optimize tự động bởi Vite
- Có thể sử dụng SVG như React components với `vite-plugin-svgr` (nếu cần)
