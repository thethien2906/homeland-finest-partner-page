# 3D Models Folder

Thư mục này chứa các file 3D models (`.glb` format) cho dự án.

## Files cần thiết:

- `vietnam-map.glb` - Bản đồ Việt Nam 3D (Scene 1)
- `vanh-non.glb` - Vành nón lá
- `la-non.glb` - Lá nón lá
- `khung-non.glb` - Khung nón lá
- `suon-non.glb` - Sườn nón lá (6 thanh)
- `quai-non.glb` - Quai nón lá

## Lưu ý:

- Tất cả models nên được scale phù hợp (khoảng 2 units)
- Models sẽ được load tự động trong code
- Đảm bảo file size hợp lý để tối ưu performance
